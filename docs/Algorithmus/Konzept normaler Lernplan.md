# Konzept normaler Lernplan

Grundprinzip

Der User hat ausreichend Zeit bis zur Klausur.

Deshalb liegt der Fokus auf:

- langfristigem Verständnis
- nachhaltigem Lernen
- mehreren Wiederholungen
- möglichst stressfreier Vorbereitung
Ziel:

Stoff verstehen → anwenden → wiederholen → sicher beherrschen

Phase 1: Grundlagen aufbauen (40%)

Ziel:

- Überblick über den Stoff bekommen
- Wissenslücken schließen
- wichtige Konzepte verstehen
Methoden:

- Vorlesungsfolien
- Skripte
- Videos
- KI-Erklärungen
- Zusammenfassungen
Anpassung durch Vorwissen

Hohes Vorwissen:

Weniger Grundlagen
Mehr Vertiefung

Wenig Vorwissen:

Mehr Grundlagen
Mehr Erklärungen

Phase 2: Vertiefung (35%)

Nachdem die Grundlagen verstanden wurden:

- Zusammenhänge erkennen
- schwierigere Themen bearbeiten
- Wissenslücken schließen
Methoden:

- Übungsaufgaben
- eigene Zusammenfassungen
- Lernkarten
- Verständnisfragen
Hier entsteht das eigentliche Fachverständnis.

Phase 3: Anwendung (15%)

Ziel:

Das Wissen aktiv nutzen.

Fokus:

- Übungsblätter
- Beispielaufgaben
- kleinere Klausuraufgaben
- Anwendungsbeispiele
Der User soll lernen:

Nicht nur wissen,
sondern anwenden können.

Phase 4: Wiederholung (10%)

Regelmäßige Wiederholung über den gesamten Lernzeitraum.

Methoden:

- Lernkarten
- Zusammenfassungen
- Formeln
- Definitionen
Je länger die Deadline, desto häufiger werden Wiederholungen eingebaut.

Wiederholungslogik

Mehr als 3 Monate

Viele Wiederholungszyklen:

Lernen
→ Wiederholen
→ Lernen
→ Wiederholen
→ Lernen
→ Wiederholen

1–3 Monate

Normale Wiederholungszyklen:

2–3 größere Wiederholungen

2–4 Wochen

Kompakter:

1–2 Wiederholungsphasen

Tagesplanung

Der Lernplan wird auf die verbleibenden Tage verteilt.

Beispiel:

Heute:
- Kapitel 3 lernen
- 5 Übungsaufgaben
- 10 Minuten Wiederholung

Der Fokus liegt auf kleinen, konstanten Lernschritten statt auf langen Lerneinheiten.

Aufgaben werden häufig verschoben

→ Stunden reduzieren
→ Kleinere Tagesziele setzen

Anpassung durch Vorwissen

Vorwissen 1–2

Mehr Zeit für:

Grundlagen
Verständnis
Erklärungen

Vorwissen 4–5

Mehr Zeit für:

Aufgaben
Vertiefung
Klausurtraining

Entscheidungslogik

StundenProTag ≤ 2
→ Normaler Lernplan

StundenProTag > 2
→ Kritischer Lernplan

Zusammenfassung

Der normale Lernplan verfolgt das Ziel:

Verstehen
→ Vertiefen
→ Anwenden
→ Wiederholen

Er nutzt die verfügbare Zeit vollständig aus, baut nachhaltiges Wissen auf und reduziert Stress durch eine gleichmäßige Verteilung der Lernlast. Der kritische Lernplan dagegen optimiert auf Effizienz und Klausurerfolg bei Zeitmangel.
