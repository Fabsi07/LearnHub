# Konzept kritischer Lernplan

Konzept: Kritischer Lernplan

Wann wird er aktiviert?

Nach der Berechnung:

Gesamtstunden = 25 × D × ((S+W+V+C)/4)

StundenProTag = Gesamtstunden / TageBisDeadline

Wenn:

StundenProTag > 2

→ Kritischer Lernplan

Grundprinzip

Der User hat nicht mehr genügend Zeit, um den gesamten Stoff optimal zu lernen.

Deshalb ändert sich das Ziel:

Normaler Lernplan

- Vollständiges Verständnis
- Nachhaltiges Lernen
- Mehrere Wiederholungszyklen
Kritischer Lernplan

- Maximale Klausurpunkte mit der verbleibenden Zeit
- Fokus auf die wichtigsten Inhalte
- Effizienz statt Vollständigkeit
Phase 1: Priorisierung (10%)

Zu Beginn werden alle Themen eingeordnet:

A-Themen

- Sehr klausurrelevant
- Kommen häufig vor
- Hohe Punktzahl
B-Themen

- Mittlere Relevanz
C-Themen

- Seltene Randthemen
Bearbeitungsreihenfolge:

A → B → C

Falls die Zeit nicht reicht, können C-Themen entfallen.

Phase 2: Kernstoff erarbeiten (50%)

Ziel:

80 % Verständnis mit möglichst wenig Zeitaufwand

Methoden:

- Zusammenfassungen
- Lernkarten
- Videos
- KI-Erklärungen
- Skripte
Kein langes detailliertes Lesen.

Phase 3: Aufgaben und Anwendung (30%)

Sobald ein Thema grob verstanden wurde:

→ direkt Aufgaben bearbeiten

Fokus auf:

- Altklausuren
- Übungsaufgaben
- typische Klausurfragen
Im kritischen Modus sind Aufgaben wichtiger als zusätzliche Theorie.

Phase 4: Wiederholung (10%)

Kurz vor der Klausur:

- Formeln
- Definitionen
- Lernkarten
- persönliche Fehlerliste
Keine neuen Themen mehr beginnen.

Tagesplanung

Auch wenn rechnerisch mehr nötig wäre:

Maximal zwei 2-Stunden-Einheiten desselben Fachs pro Tag planen.

An freien Tagen sind insgesamt bis zu drei 2-Stunden-Einheiten möglich, wenn die dritte Einheit
zu einem anderen Fach gehört.

Wenn an drei Tagen hintereinander jeweils zwei Einheiten desselben Fachs liegen, wird eine
kritische Warnung mit der Empfehlung für einen Fach- oder Pausentag angezeigt.

Der User soll nicht überfordert werden.

Statt:

Heute 5,8 Stunden lernen

lieber:

Heute:
- Thema A bearbeiten
- 2 Übungsaufgaben lösen
- Wiederholung von Thema B

Entscheidungslogik

StundenProTag ≤ 2
→ Normaler Lernplan

StundenProTag > 2
→ Kritischer Lernplan

Zusammenfassung

Der kritische Lernplan verfolgt das Ziel:

Priorisieren
→ Kernstoff lernen
→ Anwenden
→ Wiederholen

Er optimiert auf die bestmögliche Klausurleistung bei begrenzter Zeit. Der Fokus liegt auf den wichtigsten und klausurrelevanten Themen, während weniger relevante Inhalte nachrangig behandelt werden. Durch die Priorisierung und den starken Fokus auf Aufgaben und Wiederholung wird die verbleibende Lernzeit möglichst effizient genutzt.
