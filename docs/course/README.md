# Course Context

LearnHub is developed as a university project at DHBW Loerrach. The practical implementation belongs to the course "Anwendungsprojekt" and is the final assessed project result. Planning and project management run in parallel in the course "Projektmanagement".

Known milestone status:

## LearnHub Goals From Project Brief

Result goals:

- By 2026-06-26, create a demonstrable prototype of a web-based learning management app where students can centrally create, view, and edit courses, deadlines, and derived todos.
- By 2026-06-26, include at least three core screens: dashboard, calendar course/deadline overview, and todo/study planning view.
- By 2026-06-26, implement basic planning logic that prioritizes todos by deadlines, estimated effort, and subjective difficulty, then distributes them across available days.
- By 2026-06-26, investigate and prototype whether an OpenAI API feature can generate study plan suggestions from entered deadlines, capacities, and difficulty estimates.

Process goals:

- By 2026-04-27, document project idea, roles, initial tasks, and technologies in Markdown files in the GitHub repository and in the project brief. Status: completed.
- By 2026-05-21, document functional requirements as prioritized Must, Should, Could, and Won't-have requirements in the repository and align them with the lecturer.
- Until final submission on 2026-07-05, continuously document tasks, progress, and open points through GitHub Issues and GitHub Projects, and share ideas and updates in a weekly team meeting.
- By the final presentation on 2026-06-29, prepare a traceable demo of the central functions.
